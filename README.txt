== Description ==
Google campaign builder and url shortner to make short url of your campaign.


== Installation ==

1. Upload `bbilgcb.zip` to the `/wp-content/plugins/` directory
2. Extract it in this directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the `Google campaign` menu and give the required informations
